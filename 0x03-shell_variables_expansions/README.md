# 0x03. Shell, init files, variables and expansions

This project explores environment variables, shell initialization files, and expansions in Linux.

## Tasks

- **0-alias**: Create an alias `ls` that executes `rm *`
- **1-hello_you**: Print hello to the current Linux user
- **2-path**: Add `/action` to the PATH
- **3-paths**: Count the number of directories in the PATH
- **4-global_variables**: List all environment variables
- **5-local_variables**: List all local variables, environment variables, and functions
- **6-create_local_variable**: Create a local variable `BEST` with value `School`
- **7-create_global_variable**: Create a global variable `BEST` with value `School`
- **8-true_knowledge**: Print the result of adding 128 to `TRUEKNOWLEDGE`
- **9-divide_and_rule**: Print the result of `POWER` divided by `DIVIDE`
- **10-love_exponent_breath**: Display the result of `BREATH` to the power of `LOVE`
