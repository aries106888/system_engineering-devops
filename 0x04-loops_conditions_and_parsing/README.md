# 0x04. Loops, conditions and parsing

This project covers the use of loops, conditional statements, and parsing in Bash.

## Files

- `0-RSA_public_key.pub`: Public key for SSH.
- `1-for_best_school`: Displays "Best School" 10 times using a for loop.
- `2-while_best_school`: Displays "Best School" 10 times using a while loop.
- `3-until_best_school`: Displays "Best School" 10 times using an until loop.
- `4-if_9_say_hi`: Displays "Best School" 10 times, says "Hi" on the 9th iteration.
- `5-4_bad_luck_8_is_your_chance`: Loops 1-10, displays special messages for 4 and 8.
- `6-superstitious_numbers`: Loops 1-20, displays special messages for 4, 9, 17.
- `7-clock`: Displays the time every second.
- `8-for_ls`: Lists directory contents, cutting filenames.
- `9-to_file_or_not_to_file`: Checks file properties.
- `10-fizzbuzz`: FizzBuzz implementation.
- `100-read_and_cut`: Reads file line by line.
- `101-tell_the_story_of_passwd`: Parses /etc/passwd.
- `102-lets_parse_apache_logs`: Parses Apache logs.
- `103-dig_the-data`: Extracts data from CSV.
